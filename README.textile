h2. Setup

1. Clone the repository

  @git clone https://github.com/soham/oh-my-zsh.git ~/.oh-my-zsh@

2. Create a new zsh config by copying the zsh template.

  *NOTE*: If you already have a ~/.zshrc file, then just change the theme line to @export ZSH_THEME="monangik"@.

  @cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc@

3. Set zsh as your default shell:

  @chsh -s /bin/zsh@
